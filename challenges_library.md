## Формат записи

```
ID: XXX
Название: [название]
Описание: [описание с местами для чисел из base_speeds]
Тип: [спидран / лимит_перебора / один_удар / аскеза / эксперимент / антиперфекционизм / фокус / благословение]
Контекст: [любой / черновики / скетчи / презентации / ретушь / поиск / плохое_состояние / хорошее_состояние / копирайт / рутина]
Формула: [base_speeds × коэффициент]
Проверка: [артефакт / счётчик / бинарная / записанный_вывод]
```

## 🏃 Спидраны

### ID: S001
**Название:** Черновик-бомба  
**Описание:** Сделать [base_speeds × 1.4] скетчей за блок. Не доделывать, не полировать — только генерировать идеи.  
**Тип:** спидран  
**Контекст:** скетчи  
**Формула:** base_speeds(скетчи) × 1.4  
**Проверка:** счётчик  
**Статус:** активный

---

### ID: S002
**Название:** Реферальный взрыв  
**Описание:** Собрать [base_speeds × 1.3] референсов за [base_speeds × 0.8] времени. Без долгого изучения — только отбор.

**Контекст:** поиск  
**Формула:** base_speeds(поиск) × 1.3, время × 0.8

**Статус:** активный
