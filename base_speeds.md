# ⏱ Базовые скорости

> Используются Codex для расчёта чисел в челленджах.
> Обновляются по мере накопления реальных данных.
> Последнее обновление: старт системы

---

## Таблица базовых скоростей

| Тип работы | Скорость | За время |
|---|---|---|
| Сложная рекламная ретушь | 1 фото | 2.5 ч |
| Средняя ретушь (коммерческая) | 2 фото | 2.5 ч |
| Скетчи композиции | 5–6 штук | 2.5 ч |
| Поиск референсов для проекта | 1 пул | 2.5–3 ч |
| Пополнение базы референсов | 30 штук | 1.5 ч |
| Копирайт черновой | 1 текст | 1.5 ч |
| Копирайт доведение до финала | 1 текст | 1 ч |
| Сборка презентации | 12–15 слайдов | 2.5 ч |
| Отбор фотографий / деталей | 4–5 кадров | 1.5 ч |

---

## Коэффициенты для спидранов

> Применяются ТОЛЬКО к разрешённым типам задач (черновики, скетчи, презентации, поиск, рутина).

| Режим спидрана | Коэффициент |
|---|---|
| Больше результата за то же время | результат × 1.2–1.4 |
| То же за меньшее время | время × 0.7–0.8 |

⚠️ Нельзя применять оба коэффициента одновременно — только один.

## Как Codex использует эти данные

1. Берёт базовую скорость для типа задачи из контекста дня
2. Применяет коэффициент типа челленджа (спидран / аскеза / и т.д.)
3. Применяет коэффициент контекста (сложность задачи)
4. Применяет коэффициент состояния
5. Получает конкретное число для челленджа (не абстрактное «больше»)
