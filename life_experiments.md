# 🌱 Лайф-эксперименты

> Эксперименты — про жизнь вне работы. Привычки, паттерны, исследования себя.
> Каждый эксперимент — одна неделя. Запись вывода = +10 ✦.

---

## Как работает

1. Выбираешь эксперимент (или предлагаешь свой Codex)
2. Ведёшь его неделю
3. В конце недели — записываешь вывод
4. Результаты помогают Codex лучше тебя понимать

---

## Формат записи

```
### [Название эксперимента] — [Дата начала]–[Дата конца]

**Гипотеза:** [что хочешь проверить]
**Условие:** [что делаешь каждый день]
**Дни:** [ ][ ][ ][ ][ ][ ][ ]
**Вывод:** [записать в конце недели]
**Искорки:** +30 ✦ за полную неделю + 10 ✦ за запись вывода = до 40 ✦ суммарно
**Прижилась ли привычка?** Да / Нет / Частично
```

---

## Идеи для экспериментов

| Категория | Идея |
|---|---|
| Сон | Ложиться в одно время 7 дней |
| Движение | 15 минут прогулки каждый день |
| Цифровое | Первый час без телефона |
| Питание | Завтрак до начала работы |
| Паузы | Один час в день без экранов |
| Социальное | Одно живое общение в день |
| Творческое | 5 минут рисования/письма не для работы |
| Осознанность | Одна минута тишины перед первым блоком |
