# 📜 Правила игры

## Структура блока

- **Блок** = 3 часа (минимум 2.5 ч рабочих для засчитанного дня)
- **15 мин отдыха** внутри блока — после завершённого подэтапа
- **15 мин разбора входящих** — в конце блока
- **Максимум 4 блока/день**, максимум 2 подряд без часового перерыва
- **Входные условия**: только 2 — телефон в рабочем режиме + одно любое второе условие

---

## Две карты

### 🃏 Карта дня (обязательная)
- **НЕ** обязательно сложная
- Может быть: рабочий челлендж, лёгкий фан, благословение, эксперимент, тёплая карта при плохом состоянии
- **НЕ** нарушает запланированные блоки
- **1 бесплатная замена/день**
- Можно отказаться полностью → только базовые звёзды за блоки

### 🎲 Карта риска (добровольная)
- **Ставка**: 10–50 ★ (фиксированный потолок, НЕ процент)
- Ставишь **ДО** раскрытия карты
- **МОЖЕТ** нарушать планы — Codex ОБЯЗАН объяснить ЗАЧЕМ
- **Выигрыш**: ×2 ставки
- **Проигрыш**: −ставка

**Формат объяснения карты риска:**
```
⚠️ [что меняется в планах]
Почему: [одно предложение — конкретная причина из контекста]
Не хочешь — не бери, ставка вернётся.
```

**Допустимые причины для карты риска:**
- Хороший поток + горящий проект
- Отдыхала 2+ дней, энергия восстановлена
- Высокая энергия / отличное состояние
- Близкий дедлайн + высокая готовность

**НЕдопустимые причины:**
- «Просто так»
- «Надо больше работать»
- «Ты ленивая»
- Любое давление через вину

## Экономика — звёзды ★

### Заработок

| Действие | Звёзды |
|---|---|
| Завершённый блок с артефактом | 10 ★ |
| Карта дня выполнена | 10–30 ★ (по формуле) |
| Бонусное условие карты | +5–15 ★ |
| Карта риска выиграна | ×2 ставки |
| Серия карта дня 3 дня | +20 ★ |
| Серия карта дня 5 дней | +50 ★ |
| Серия карта дня 7 дней | +100 ★ |
| Недельный челлендж 3 дня | +15 ★ |
| Недельный челлендж 5 дней | +35 ★ |
| Недельный челлендж 7 дней | +70 ★ |

⚠️ Блок БЕЗ артефакта (чистое изучение) = 0 ★

### Траты (послабления = личная ответственность)

| Послабление | Стоимость |
|---|---|
| Бокал алкоголя | 40 ★ |
| Незапланированный выходной | 30 ★ |
| Работа после полуночи | 30 ★ (за ночь, НЕ за час) |
| Срыв собственного дедлайна | 15 ★ |

> Срыв дедлайна = когда **могла** сдать вовремя при нормальной работе, но не сдала.

## Крупная награда

- **Горизонт**: ~4 недели (1 месяц)
- Замену можно произвести в любой момент
- Если подарили запланированное → «возврат», покупаешь другое на ту же сумму
- Если не заработала за месяц → приз сгорает, но стартовые бонусы переносятся

**Расчёт целевой суммы:**
- Средний день: ~35 ★
- Средняя неделя: ~225 ★
- Хорошая неделя: ~300 ★
- Послабления: ~80–100 ★/месяц
- **Целевая сумма подбирается = ~4 недели нормальной работы**

---

## Формула расчёта звёзд за челлендж

Звёзды **НЕ прописаны жёстко** в челленджах. Codex считает по формуле:

```
Звёзды = БазоваяСложность × КоэфКонтекста × КоэфСостояния
```

| Параметр | Значение |
|---|---|
| Спидран | 1.0 |
| Аскеза | 1.2 |
| Один удар | 1.3 |
| Эксперимент | 0.8 |
| Антиперфекционизм | 1.0 |
| Благословение | фиксированное |
| Сложная задача | ×1.3 |
| Средняя задача | ×1.0 |
| Простая задача | ×0.8 |
| Плохое состояние | ×1.2 |
| Нормальное состояние | ×1.0 |
| Отличное состояние | ×0.9 |

- Результат **округляется до 5**
- Минимум **10 ★**, максимум **30 ★** за карту дня

---

## Антиинфляционный механизм

- Codex ведёт целевой план: **~250 ★/неделю** при нормальной работе
- Если фактический темп >120% от плана → награды снижаются на 10%
- Если <80% → **НЕ** увеличиваются (не подачки!)
- Codex предупреждает о перекосе
- Если 3 дня подряд разница >30% → пересмотр формулы

## Искорки ✦ (лайф-валюта)

### Заработок

| Действие | Искорки |
|---|---|
| Завершённый лайф-эксперимент (полная неделя) | 30 ✦ |
| Запись вывода эксперимента | +10 ✦ |
| Серия 2 недели | +20 ✦ |
| Серия 3 недели | +30 ✦ + привычка «прижилась» |

### Траты (игровые привилегии)

| Привилегия | Стоимость |
|---|---|
| Вторая замена карты дня | 20 ✦ |
| Карта благословения по запросу | 40 ✦ |
| Снятие штрафа за послабление | 50 ✦ |
| Разморозка серии (после 1 дня пропуска) | 30 ✦ |
| Право выбрать недельный челлендж | 15 ✦ |

**Замораживание**: если за неделю <6 рабочих блоков → искорки нельзя тратить (но копятся).

⚠️ **ЗАПРЕЩЕНО** в наградах и тратах: еда, кофе, базовые потребности, ванна, маска, прогулка — это ЖИЗНЬ, не награда.

---

## Недельные челленджи

- Выпадает в начале недели **случайно**
- **Серии**: 3 дня +15 ★, 5 дней +35 ★, 7 дней +70 ★
- Порвала серию → начинаешь с нуля, но челлендж остаётся до конца недели

---

## Калибровка сложности

| Условие | Действие |
|---|---|
| 3 выполнения подряд | Новая база (максимум +20%/месяц) |
| 2 невыполнения подряд | Коэффициент −0.1 |
| 3 невыполнения подряд | Челлендж убирается из ротации до пересмотра |

---

## Карта дня — антипотворство

- При **плохом состоянии**: НЕ «сделай одно и отдыхай». А «выпей таблетку, подожди 45 минут, потом один важный блок. Получилось — звёзды.»
- Система **НЕ подстраивается под отставание**
- Не даёт лёгкие задания за много звёзд при отставании
- **Честная игра** — всегда

## Типы челленджей

| Тип | Описание | Коэффициент | Ограничения |
|---|---|---|---|
| Спидран | Больше результата или меньше времени | 1.0 | ТОЛЬКО черновики, скетчи, презентации, поиск, рутина. НИКОГДА глубокая ретушь/съёмка/стратегия |
| Лимит перебора | Ограничение попыток/вариантов/минут | 1.0 | Против перфекционизма |
| Один удар | Глубокий фокус на одной задаче | 1.3 | Точки А и Б по контексту |
| Аскеза | Убрать привычный костыль | 1.2 | С записью момента и причины если сорвалась |
| Эксперимент | Проверка гипотезы о своей работе | 0.8 | Всегда с записанным выводом |
| Антиперфекционизм | Стоимость ошибки, ценность «достаточно» | 1.0 | — |
| Фокус | Концентрация на одном типе задачи | 1.0 | — |
| Благословение | Баланс, фан, восстановление | фиксированное | При выгорании или для удовольствия |
