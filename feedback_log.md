# 📝 Лог обратной связи

> Сюда записывается ВСЯ обратная связь пользователя — без фильтрации и интерпретации.
> Эмоции, рефлексия, недоумения, предложения, реакции на карты, мысли о системе.
> Codex записывает СРАЗУ, не дожидаясь итога дня.
> Этот файл НЕ сжимается и НЕ удаляется — это сырой материал для ежемесячного аудита через Opus.

---

## Как записывать

Codex добавляет запись в формате:

```
### [Дата] [Время/контекст]

**Ситуация:** [что делала, какая карта, какой блок]
**Цитата:** «[дословно что сказала пользователь]»
**Тип:** [эмоция / рефлексия / предложение / недоумение / критика / похвала / вопрос]
**Действие:** [что Codex сделал в ответ, если что-то сделал]
```
