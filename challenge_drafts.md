# 📝 Черновики новых челленджей

> Генерируются Codex автоматически после каждого блока.
> ПРАВИЛО: каждый черновик — ДРУГОЙ тип, чем оригинал, который его породил.
> Спидран → порождает НЕ спидран. Эксперимент → порождает НЕ эксперимент.

---

## Формат черновика

```
### ЧЕРНОВИК-[ДАТА]-[порядковый номер]

**Источник:** [ID оригинала или «новая идея»]
**Оригинальный тип:** [тип]
**Новый тип:** [ДРУГОЙ тип]
**Название:** [название]
**Описание:** [черновик описания]
**Контекст:** [предполагаемый контекст]
**Проверка:** [предполагаемый тип проверки]
**Статус:** черновик
**Одобрено:** нет / да — [дата]
```
