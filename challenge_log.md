# 📓 Дневник челленджей

> Заполняется Codex автоматически после каждого завершённого блока.
> Пользователь добавляет только «отзыв» и «оценку» через чат.

---

## Шаблон записи

```
### [Дата] — [ID челленджа] [Название]

**Тип:** [тип]  
**Выполнено:** Да / Нет  
**Артефакт:** [описание или ссылка]  
**Звёзды получено:** ★  
**Отзыв:** [что сказала пользователь]  
**Что сработало:** [заполняется пользователем или Codex по отзыву]  
**Что не сработало:** [заполняется пользователем или Codex по отзыву]  
**Оценка:** [1–5]  
**Заметка Codex:** [наблюдение для калибровки]
```
