# 📅 Недельные челленджи

> Выпадает в начале недели случайно. Codex выбирает из этого пула.
> Серии: 3 дня +15 ★, 5 дней +35 ★, 7 дней +70 ★
> Порвала серию → начинаешь с нуля, но челлендж остаётся до конца недели.

---

## Формат

```
ID: W-XXX
Название: [название]
Описание: [привычка на неделю]
Тип: [поведенческий / продуктивность / качество / баланс / исследование]
Проверка: [бинарная ежедневная / счётчик / записанный_вывод]
Статус: [активный / отключён]
```
